# Online Library Management System

A complete system for managing public, university, and personal libraries that enables member registration, book borrowing, and book returns.

## Main Features

### 1. User Management

- User registration and login (members and library administrators)
- Role assignment (administrator, regular member)
- Viewing the history of borrowed and returned books

### 2. Book Management

- Ability to add a new book along with its details (title, author, genre, publication date, number of available copies)
- Search and filtering by author, genre, book title, and publication year
- Ability for the library administrator to edit book information

### 3. Book Borrowing and Return System

- Members can request the books they want
- Setting the loan duration and book return date
- Ability to reserve a book if all copies have been borrowed
- Assigning a fine for late book returns

### 4. Notifications and Reminders

- Sending emails or text messages to remind members of book return dates
- Sending notifications for pending reservations

### 5. Administrative Dashboard

- Displaying the number of borrowed books and the available stock of each book
- Viewing delays and users with fines
- Generating reports on book popularity and member performance

### 6. Rating and Review System

- Members can rate books and submit their reviews
- Displaying user reviews and ratings to other members

### 7. Online Fine Payment

- In the event of a late book return, members can pay the fine online

## Extra Credit Items

- An intelligent recommendation system that suggests similar books based on books that have been read or rated
- Recommending related books based on users' interests
- Ability to view a list of “Recommended Books for You” on the user dashboard
